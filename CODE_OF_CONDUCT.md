# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be respectful. No harassment, no slurs, no doxxing. Disagreement is fine; personal attacks aren't.

## Scope

Applies anywhere this project lives — issues, PRs, discussions, code comments, email, social media when representing the project.

## Enforcement

The maintainer (`@iamjohnnymac`) handles reports. Email `macca.mck+conduct@gmail.com` for anything you don't want public. Consequences range from a private warning to a permanent ban from the project.
